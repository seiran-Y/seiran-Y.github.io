lxml HTML API 速查

本文按功能分类整理 lxml 在 HTML 解析场景下的常用 API，包括 HTML 解析、XPath 选择、文本和属性提取、节点遍历、节点修改与序列化输出。

lxml 是 Python 里常用的 HTML/XML 解析库。爬虫场景里主要用 lxml.etree.HTML() 把网页字符串解析成节点树，然后用 XPath 提取内容。

常用导入：

1. 解析 HTML

etree.HTML(text, parser=None, base_url=None)

把 HTML 字符串解析成 Element 对象。

参数：

  • text：HTML 字符串，类型通常是 str 或 bytes
  • parser：可选，自定义解析器
  • base_url：可选，设置基础 URL，处理相对链接时有用

示例：

etree.fromstring(text, parser=None)

从字符串解析 XML 或 XHTML。普通网页优先用 etree.HTML()，因为网页经常不规范。

参数：

  • text：XML/XHTML 字符串或 bytes
  • parser：解析器

2. XPath 选择节点

element.xpath(path, namespaces=None, extensions=None, smart_strings=True, **kwargs)

执行 XPath 查询。

参数：

  • path：XPath 表达式
  • namespaces：命名空间映射，HTML 爬虫一般不用
  • smart_strings：是否返回带上下文的字符串，默认 True
  • **kwargs：XPath 变量

示例：

3. 提取文本

xpath("text()")

只取当前节点的直接文本，不包含子标签里的文本。

xpath("string(.)")

取当前节点及所有子节点的完整文本。

element.text

当前节点开始标签后的直接文本。

element.tail

当前节点结束标签后面的尾随文本。

4. 提取属性

xpath("@attr")

通过 XPath 提取属性。

element.get(key, default=None)

从节点对象获取属性。

参数：

  • key：属性名
  • default：属性不存在时返回的默认值

5. 遍历节点

element.iter(tag=None, *tags)

深度遍历当前节点下的所有子节点。

参数：

  • tag：只遍历指定标签；不传则遍历全部

element.iterchildren(tag=None, reversed=False)

遍历直接子节点。

参数：

  • tag：筛选标签名
  • reversed：是否反向遍历

element.getparent()

获取父节点。

element.getchildren()

获取直接子节点列表。新代码更推荐用 list(element)。

6. 修改节点

element.set(key, value)

设置属性。

参数：

  • key：属性名
  • value：属性值

element.text = value

修改节点文本。

element.append(child)

添加子节点。

参数：

  • child：Element 对象

element.remove(child)

删除子节点。

7. 序列化输出

etree.tostring(element, encoding=None, method="xml", pretty_print=False)

把节点树转成字符串或 bytes。

参数：

  • element：节点对象
  • encoding：编码；传 "unicode" 返回字符串
  • method：输出方式，HTML 用 "html"
  • pretty_print：是否格式化换行

8. HTMLParser 常用配置

etree.HTMLParser(encoding=None, recover=True, remove_blank_text=False)

自定义 HTML 解析器。

参数：

  • encoding：指定网页编码，例如 "utf-8"、"gbk"
  • recover：遇到错误是否尽量修复，默认 True
  • remove_blank_text：是否移除空白文本节点

10. 常见坑

  • XPath 下标从 1 开始，不是 Python 的 0
  • //div[@class='a'] 是精确匹配 class，多个 class 时容易匹配不到
  • 多 class 推荐 contains(@class, 'xxx')
  • text() 只取直接文本，正文常用 string(.)
  • XPath 返回通常是列表，取第一个前要判断是否为空
  • 网页如果是 JS 渲染出来的，lxml 只能解析请求到的 HTML，不能执行 JS
