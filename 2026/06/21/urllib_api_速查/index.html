urllib API 速查

urllib 是 Python 标准库里的 URL 和 HTTP 工具包，不需要安装。它适合写轻量请求、URL 参数处理、下载文件。复杂爬虫里常用 requests，但理解 urllib 对看源码和处理基础请求很有帮助。

常用导入：

1. urllib.request：发送请求

urlopen(url, data=None, timeout=socket._GLOBAL_DEFAULT_TIMEOUT, *, cafile=None, capath=None, cadefault=False, context=None)

打开一个 URL，返回 HTTPResponse 对象。

参数：

  • url：字符串 URL，或 Request 对象
  • data：请求体；传 bytes 时会变成 POST 请求
  • timeout：超时时间，单位秒
  • context：SSL 上下文，HTTPS 证书相关时使用

GET 示例：

Request(url, data=None, headers={}, origin_req_host=None, unverifiable=False, method=None)

构造请求对象，用于添加 headers、指定 method。

参数：

  • url：请求地址
  • data：请求体，必须是 bytes 或 file-like 对象
  • headers：请求头字典
  • method：请求方法，例如 "GET"、"POST"

示例：

POST 表单请求

JSON POST 请求

urlretrieve(url, filename=None, reporthook=None, data=None)

下载 URL 到本地文件。简单下载可以用，但新代码更推荐自己用 urlopen() 读取后写入文件。

参数：

  • url：下载地址
  • filename：保存路径
  • reporthook：进度回调函数
  • data：POST 数据

2. HTTPResponse 常用方法

resp.read(amt=None)

读取响应体。

参数：

  • amt：读取的字节数；不传则读取全部

resp.status

HTTP 状态码。

resp.headers

响应头。

resp.geturl()

获取最终 URL，遇到重定向时很有用。

3. urllib.parse：URL 编码和解析

urlencode(query, doseq=False, safe="", encoding=None, errors=None, quote_via=quote_plus)

把字典或二元组列表编码成 query string。

参数：

  • query：字典或键值对列表
  • doseq：值是列表时是否展开
  • safe：不转义的字符
  • encoding：编码，默认 UTF-8
  • errors：编码错误处理方式
  • quote_via：使用的 quote 函数

列表参数：

quote(string, safe="/", encoding=None, errors=None)

对 URL 的某个部分进行百分号编码。

参数：

  • string：要编码的字符串
  • safe：保留不编码的字符，默认保留 /
  • encoding：编码
  • errors：错误处理

unquote(string, encoding="utf-8", errors="replace")

把百分号编码还原。

参数：

  • string：已编码字符串
  • encoding：解码编码
  • errors：错误处理

urlparse(url, scheme="", allow_fragments=True)

拆分 URL。

参数：

  • url：完整 URL
  • scheme：默认协议
  • allow_fragments：是否解析 #fragment

parse_qs(qs, keep_blank_values=False, strict_parsing=False, encoding="utf-8", errors="replace", max_num_fields=None, separator="&")

解析 query string，返回字典，值是列表。

参数：

  • qs：query string，不包含 ?
  • keep_blank_values：是否保留空值
  • strict_parsing：格式错误时是否抛异常
  • encoding：解码编码
  • errors：错误处理
  • max_num_fields：最大字段数
  • separator：分隔符

urljoin(base, url, allow_fragments=True)

把相对链接合并成绝对链接。

参数：

  • base：基础 URL
  • url：相对或绝对 URL
  • allow_fragments：是否允许 fragment

urlunparse(parts)

把拆开的 URL 部分重新组合。

参数：

  • parts：6 个元素，顺序是 scheme, netloc, path, params, query, fragment

4. urllib.error：异常处理

HTTPError

HTTP 状态码错误，例如 404、403、500。

常用属性：

  • code：状态码
  • reason：原因
  • headers：响应头

URLError

URL 打开失败，例如 DNS 错误、连接失败、超时。

常用属性：

  • reason：失败原因

5. urllib.robotparser：robots.txt

RobotFileParser(url="")

读取并判断 robots.txt 规则。

常用方法：

  • set_url(url)：设置 robots.txt 地址
  • read()：读取 robots.txt
  • can_fetch(useragent, url)：判断某 user-agent 是否可以抓取 URL

7. urllib 和 requests 的区别

  • urllib：标准库，无需安装，但写法偏底层
  • requests：第三方库，API 更友好，爬虫更常用
  • 如果只是学习标准库或写无依赖脚本，用 urllib
  • 如果是实际爬虫/API 项目，通常用 requests

8. 常见坑

  • POST 的 data 必须是 bytes，不能直接传 dict
  • 中文参数要用 urlencode() 或 quote() 编码
  • urlopen().read() 返回 bytes，需要 .decode()
  • 要设置超时 timeout，否则网络卡住时可能一直等
  • 复杂 cookie/session 登录场景不适合纯 urlopen，用 requests.Session() 更方便
