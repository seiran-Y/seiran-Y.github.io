pathlib API 速查

本文按文件操作场景整理 pathlib 的常用 API，包括路径构造、创建、读写、遍历、删除、移动、复制、文件名处理和路径转换。

pathlib 是 Python 标准库里的路径操作模块。它用对象方式处理文件和目录，比 os.path 更直观。

常用导入：

1. 创建路径对象

Path(*pathsegments)

创建路径对象。

参数：

  • *pathsegments：路径片段，可以传一个完整路径，也可以传多个片段

Path.cwd()

获取当前工作目录。

Path.home()

获取当前用户主目录。

2. 路径拼接

/ 运算符

用 / 拼接路径，跨平台。

path.joinpath(*other)

拼接路径。

参数：

  • *other：一个或多个路径片段

3. 判断路径状态

path.exists()

判断路径是否存在。

path.is_file()

判断是否是文件。

path.is_dir()

判断是否是目录。

path.is_absolute()

判断是否是绝对路径。

4. 创建目录

path.mkdir(mode=0o777, parents=False, exist_ok=False)

创建目录。

参数：

  • mode：权限，Windows 下通常不用管
  • parents：是否创建不存在的父目录
  • exist_ok：目录已存在时是否不报错

5. 创建文件

path.touch(mode=0o666, exist_ok=True)

创建空文件。

参数：

  • mode：权限
  • exist_ok：文件已存在时是否不报错

6. 读取文本

path.read_text(encoding=None, errors=None)

读取文本文件，返回字符串。

参数：

  • encoding：编码，例如 "utf-8"、"utf-8-sig"、"gbk"
  • errors：解码错误处理方式，例如 "ignore"、"replace"

处理 Windows 记事本友好文件：

7. 写入文本

path.write_text(data, encoding=None, errors=None, newline=None)

写入文本。会覆盖原文件。

参数：

  • data：要写入的字符串
  • encoding：编码
  • errors：编码错误处理方式
  • newline：换行符控制，例如 "\n"

写入带 BOM 的 UTF-8，方便 Windows 记事本识别中文：

8. 读取和写入二进制

path.read_bytes()

读取二进制文件。

path.write_bytes(data)

写入二进制文件。会覆盖原文件。

参数：

  • data：bytes 数据

9. 打开文件

path.open(mode="r", buffering=-1, encoding=None, errors=None, newline=None)

像内置 open() 一样打开文件。

参数：

  • mode：打开模式，如 "r"、"w"、"a"、"rb"、"wb"
  • buffering：缓冲策略
  • encoding：文本编码
  • errors：错误处理
  • newline：换行符控制

10. 遍历目录

path.iterdir()

遍历目录下的直接子项。

path.glob(pattern)

按模式搜索当前目录，不递归。

参数：

  • pattern：匹配规则，例如 "*.txt"、"chapter*.html"

path.rglob(pattern)

递归搜索子目录。

参数：

  • pattern：匹配规则

11. 删除文件和目录

path.unlink(missing_ok=False)

删除文件或符号链接。

参数：

  • missing_ok：路径不存在时是否不报错，Python 3.8+

path.rmdir()

删除空目录。目录必须为空。

删除非空目录需要 shutil.rmtree()：

12. 移动和重命名

path.rename(target)

重命名或移动文件/目录。

参数：

  • target：目标路径，字符串或 Path

移动到目录：

path.replace(target)

替换目标路径。目标存在时会覆盖。

参数：

  • target：目标路径

13. 复制文件

pathlib 本身没有复制方法，通常配合 shutil.copy2()。

shutil.copy2(src, dst)

复制文件，并尽量保留元数据。

参数：

  • src：源文件路径
  • dst：目标文件或目标目录

14. 文件名、后缀、父目录

path.name

文件名，包含后缀。

path.stem

文件名，不包含最后一个后缀。

path.suffix

最后一个后缀。

path.suffixes

所有后缀。

path.parent

父目录。

path.parents

所有上级目录。

15. 修改路径组成

path.with_name(name)

替换文件名。

参数：

  • name：新文件名

path.with_suffix(suffix)

替换后缀。

参数：

  • suffix：新后缀，必须带点，例如 ".md"

path.with_stem(stem)

替换主文件名，Python 3.9+。

参数：

  • stem：新的主文件名

16. 绝对路径和相对路径

path.resolve(strict=False)

返回绝对路径，并解析 ..、.。

参数：

  • strict：路径不存在时是否报错

path.absolute()

返回绝对路径，不一定解析符号链接。

path.relative_to(*other, walk_up=False)

计算相对路径。

参数：

  • *other：基准路径
  • walk_up：Python 3.12+，允许用 .. 向上计算

17. 文件信息

path.stat()

获取文件状态信息。

常用属性：

  • st_size：文件大小，单位 bytes
  • st_mtime：最后修改时间戳
  • st_ctime：创建时间或元数据变更时间，取决于系统

19. 常见坑

  • write_text() 会覆盖原文件
  • unlink() 只能删文件，不能删目录
  • rmdir() 只能删空目录
  • rename() 移动到不存在的父目录会失败，需要先 mkdir()
  • Windows 文件名不能包含 \ / : * ? " < > |
  • Path 对象传给大多数现代库都可以直接用；老库不支持时用 str(path)
