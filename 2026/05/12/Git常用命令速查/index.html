Git 常用命令速查

分类： 编程基础
标签： Git工具开发效率

日期：2026-05-12 17:30:00

Git 是程序员最常用的版本控制工具。它可以记录项目的每一次变化，让你放心修改、回退、分支开发，并把代码同步到 GitHub 等远程仓库。

这篇文章按功能分块整理 Git 的常用命令、参数和典型使用场景，适合作为日常开发时的速查表。

一、Git 的核心概念

理解 Git，先记住四个区域：

区域 含义
工作区 你正在编辑的项目文件夹
暂存区 准备提交的一组修改
本地仓库 已经提交保存的版本历史
远程仓库 GitHub、Gitee 等平台上的仓库

十二、命令速查表

功能 命令
初始化仓库 git init
克隆仓库 git clone <url>
查看状态 git status
查看修改 git diff

提交修改 git commit -m "message"
查看历史 git log --oneline

临时保存 git stash
恢复 stash git stash pop
