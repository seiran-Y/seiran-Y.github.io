基于GitHub+hexo的博客搭建(我的第一篇博客)

前置准备：

  • node.js：提供运行 javascript 的环境

    • npm 包管理器
    • 检测：node -v
      npm -v

  • git：版本控制系统，面向代码文件的版本控制

    • 检测：git --version

  • github：新建仓库命名规则 [账号名.github.io]

    • 每个账号只能有一个这样的用户/组织主页

    • GitHub 会自动将博客放到该网址

    • 注意：

      • 新建仓库是完全空的，没有 main 分支

      • 自行创建 README 后自动创建 main

      • 在 Build and deployment 中设置为

        • Source = Deploy from a branch
        • Branch = main 再选 /（root）

  • hexo：博客框架
