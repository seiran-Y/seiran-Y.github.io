3. GitHub

用于托管博客的仓库。

  • 新建仓库命名规则：账号名.github.io
  • 每个账号只能有一个这样的用户/组织主页
  • GitHub 会自动将博客部署到该网址

注意：

  • 新建仓库时是空的，没有 main 分支

  • 创建 README 后会自动生成 main 分支

  • 在 Pages 配置中需设置：

    • Source = Deploy from a branch
    • Branch = main（目录选择 /root）
