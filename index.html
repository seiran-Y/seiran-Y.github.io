自动方式

步骤说明：

  1. 安装 locales：提供语言环境生成工具
  2. sed 命令：自动去掉 /etc/locale.gen 文件中 # zh_CN.UTF-8 UTF-8 前的 #
  3. locale-gen：根据 /etc/locale.gen 生成中文语言包
  4. update-locale：设置系统默认语言为 zh_CN.UTF-8

前置准备：

  • node.js：提供运行 javascript 的环境

    • npm 包管理器
    • 检测：node -v
      npm -v

  • git：版本控制系统，面向代码文件的版本控制

    • 检测：git --version

  • github：新建仓库命名规则 [账号名.github.io]

    • 每个账号只能有一个这样的用户/组织主页

    • GitHub 会自动将博客放到该网址

    • 注意：

      • 新建仓库是完全空的，没有 main 分支

      • 自行创建 README 后自动创建 main

      • 在 Build and deployment 中设置为

        • Source = Deploy from a branch
        • Branch = main 再选 /（root）
